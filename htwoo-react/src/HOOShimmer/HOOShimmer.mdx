# HOOShimmer

## Item Interface

The `theme` property and the `shape` property have exported enums to help set the values. See defintions below:

```ts
export enum HOOShimmerTheme {
  "None",
  "Neutral",
  "Primary",
  "Fancy"
}

export enum HOOShimmerShape {
  "Container",
  "Circle",
  "Row",
  "Square",
  "Image1:1",
  "Image16:9",
  "Image16:10"
}
```

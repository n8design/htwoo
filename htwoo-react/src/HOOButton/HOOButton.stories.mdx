# HOOButton

Predefined enum defined for the supported button types:
```ts
export enum HOOButtonType {
  "Icon",
  "Primary",
  "Standard",
  "HyperlinkPrimary",
  "HyperlinkStandard",
  "CompoundPrimary",
  "CompoundStandard"
}
```

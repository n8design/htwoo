# HOOAvatar

Predefined enum defined for the supported avatar sizes:
```ts
export enum HOOAvatarSize {
  Px16 = "16",
  Px24 = "24",
  Px32 = "32",
  Px40 = "40",
  Px48 = "48",
  Px64 = "64",
  Px72 = "72",
  Px96 = "96",
}
```

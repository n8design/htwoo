# HOOProgressStepBar

A HOOProgressStepBar component will isolate the child components passed to it looking for a HOOProgressBar and all instances of HOOProgressStep. The HOOProgressBar will be rendered and then the array of HOOProgressStep components below it. All other child components will be ignored.

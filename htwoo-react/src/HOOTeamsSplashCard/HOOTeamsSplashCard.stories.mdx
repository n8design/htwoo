# HOOTeamsSplashCard

This example shows using a `HOOSplashCardHeader`, `HOOSplashCardTitle`, `HOOSplashCardDesc`, and `HOOSplashCardFooter` (`HOOButton`) components as the children of the `HOOTeamsSplashCard` component.

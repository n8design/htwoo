# HOODropDown

## Item Interface

The `options` property should be of type `IHOODropDownItem`, or `IHOODropDownGroup` which then contains
groupItems which is an array of `IHOODropDownItem` values. See defintions below:

```ts
export interface IHOODropDownItem {
  key: string | number;
  text: string;
  disabled: boolean;
}

export interface IHOODropDownGroup {
  groupName: string;
  groupItems: IHOODropDownItem[];
}
```

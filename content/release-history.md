## v0.4.0
- New component scss file that encapsulate all previous components
```tsx
@import 'node_modules/@n8d/htwoo-core/lib/components/all'
```
- Add new shimmer image placeholders for various ratios - [squared](https://htwoo.io/htwoo-core/?p=atoms-shimmer-squared), [16:9](https://htwoo.io/htwoo-core/?p=atoms-shimmer-img-16x9), [16:10](https://htwoo.io/htwoo-core/?p=atoms-shimmer-img-16x10) images
- Microsoft Teams [Toolbar](https://htwoo.io/htwoo-core/?p=molecules-teams-toolbar) and first draft of Microsoft Teams [Dashboard](https://htwoo.io/htwoo-core/?p=templates-teams-dashboard)
- Template implementation of a responsive [Card Grid]()https://htwoo.io/htwoo-core/?p=templates-card-grid
- Text overlfow fix for card layouts
- Adjustment of image borders
- **EXPERIMENTAL:** JavaScript supports now AMD, CommonJS and UMD in `dist`folder of package. untransformed JavaScript still remains in the `lib` folder.

**Documentation:** [Component Library and Style Guide - v0.4.0](/htwoo/htwoo-core)
